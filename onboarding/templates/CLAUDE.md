# Welcome

This is a Claude Code workspace template. To get started, run the onboarding skill:

> "Run the onboarding skill"

Claude will walk you through setting up your identity, Obsidian vault, MCP connections, and scheduled tasks.

## Before you start

1. Copy `.mcp.json.template` to `.mcp.json` and fill in your tokens
2. Open Claude Code in this directory
3. Say "Run the onboarding skill"

## What gets set up

- `config/user-context.md` — your identity, timezone, and connected tools
- `vault/CLAUDE.md` — your org context, workflow, and preferences
- `vault/projects/_index.md` — your active projects list
- Scheduled tasks (morning briefing, EOD wrap-up, weekly digest) via launchd
